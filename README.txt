localStorage - сохраняет данные в браузере на стороне клиента. Эти данные сохраняются в виде строк
и доступны только для веб-приложения, которое их сохранило (в пределах одного домена, протокола и порта)




Не имеют ограничения по времени хранения
Хранятся до тех пор, пока:
1. Пользователь вручную не удалит их через настройки браузера.
2. Программа(js файл) явно не очистит данные через localStorage.removeItem() или localStorage.clear().


Не подходит для:
1. Чувствительных данных (пароли, токены).
2. Больших объёмов данных (ограничение в 5 МБ).


localStorage идеально подходит для:
1. Хранения пользовательских настроек (например, тёмная тема).
2. Кэширования данных для офлайн-приложений (например, результаты формы).
3. Хранения идентификаторов или токенов (не чувствительных).
